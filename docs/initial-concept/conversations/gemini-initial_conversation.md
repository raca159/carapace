## 3. C# Architecture: Procedural Generation Engine

To code this cleanly in C#, you can use a data-driven approach where mutations are components or objects applied to a base `Enemy` class during world generation.

### The Code Architecture Concept

```csharp
public enum MutationTier { Minor, Major, Apex }

public interface IMutation
{
    string Name { get; }
    MutationTier Tier { get; }
    string VisualDescription { get; }
    void ApplyStats(EnemyStats stats, float ageMultiplier);
    void TriggerOnAttack(Entity attacker, Entity target);
}

public class SonicCavitationMutation : IMutation
{
    public string Name => "Sonic Cavitation";
    public MutationTier Tier => MutationTier.Major;
    public string VisualDescription => "an oversized, heavily reinforced crushing claw that twitches rhythmically";

    public void ApplyStats(EnemyStats stats, float ageMultiplier)
    {
        stats.MeleeDamage += 10 * ageMultiplier;
        stats.ArmorPenetration += 0.15f * ageMultiplier;
    }

    public void TriggerOnAttack(Entity attacker, Entity target)
    {
        // 20% chance to stun the player via sonic boom
        if (Random.Shared.NextDouble() < 0.20)
        {
            target.ApplyStatusEffect(StatusEffect.Stunned, duration: 2);
            GameLog.Add($"{attacker.Name} snaps its claw, releasing a localized sonic boom!");
        }
    }
}

```

### The Procedural Factory

When the world generator populates a dungeon room or a vampire castle, it rolls an age variable for the boss.

```csharp
public class EnemyFactory
{
    public Enemy GenerateProceduralBoss(int depthLevel)
    {
        Enemy boss = new Enemy();
        // Deeper dungeons = older, more terrifying horrors
        int baseAge = Random.Shared.Next(50, 200) * depthLevel; 
        boss.Age = baseAge;
        
        float ageMultiplier = 1.0f + (baseAge / 100.0f); // Every century adds 100% power
        
        // Pick mutations from the biological library based on age/tier
        List<IMutation> rolledMutations = MutationLibrary.RollMutationsForAge(baseAge);
        
        foreach(var mutation in rolledMutations)
        {
            boss.Mutations.Add(mutation);
            mutation.ApplyStats(boss.BaseStats, ageMultiplier);
        }
        
        // Construct the ASCII/Terminal description dynamically for the LLM API to read!
        boss.UpdateDescription(); 
        
        return boss;
    }
}

```

## 1. The Splicing Loop & Probability Mechanics

When you defeat an Elite, Unique, or Legendary enemy, there is a chance they leave behind a **Viable Tissue Sample** (e.g., *"Pistol Shrimp Claw Fragment (Major)"*). To use it, you must carry it back to a functional pre-collapse terminal.

The process checks against three variables: **Sample Quality**, **Player Bio-Tech Skill**, and **Current Genetic Load**.

```

              +---------------+---------------+
              |                               |
       (Roll Success %)               (Roll Failure %)
```

### The Mutation Outcomes

1. **Critical Success:** The gene adapts perfectly. You gain the beneficial modifier (e.g., *Sonic Cavitation* active ability) with zero downside.
2. **Stable Success:** You gain the beneficial modifier, but your maximum DNA Stability permanently drops by a fixed percentage to accommodate the new strain.
3. **Splice Malapty (Failure):** The foreign DNA aggressively overrides your human genome. You do *not* get the desired skill; instead, you contract a **Genetic Malapty**—a biological defect that mimics the downside of that animal's traits.

## 4. World Reaction & LLM Integration

This is where the terminal graphics and the LLM API create amazing emergent gameplay. As you gain more mutations (both successful and failed), you acquire **Disfigurement Points**.

Your ASCII/Pixel sprite physically transforms. Your `@` symbol might change color to an oily green, or gain trailing characters representing extra limbs (`@@`).

### The NPC Dialogue Shift

When you interact with standard human NPCs in towns, the game updates your metadata profile sent to the LLM API.

**Early Game Profile (Pure Human):**

```json
{ "player_appearance": "Wearing ancient fabric armor, face clear and human." }

```

* *NPC Guard:* "Welcome to the keep, Hunter. The King is waiting for you."

**Late Game Profile (Highly Mutated):**

```json
{ "player_appearance": "Left arm is a massive, twitching chitinous claw. Skin is shifting in blue and purple iridescent cuttlefish patterns. Compound eyes flicker in the shadows." }

```

* *NPC Guard (AI Generated Output):* "Halt... *What* are you? Stand back from the gates! You look more like the things that drag our children into the trenches than a man. Guards, raise your crossbows!"

### The Vampire Faction Synergy

Conversely, if you become heavily mutated, the **Sanguine Elite** and **Familiars** might stop attacking you on sight. They will recognize you as a fellow "ascended" being.

* You can walk into Vampire-controlled cities unbothered.
* The AI puppet-masters might offer you quests to *betray* the human kingdoms, giving you access to forbidden alien gene-splicing vaults that humans can't operate.

## C# Implementation Blueprint for Splicing

You can structure this within your player state system using an underlying array of genetic markers:

```csharp
public class PlayerGenome
{
    public float HumanDNAPurity { get; set; } = 1.0f; // 1.0 = 100% human
    public List<IMutation> ActiveSplices { get; set; } = new List<IMutation>();
    public List<Malapty> ActiveMalapties { get; set; } = new List<Malapty>();
    public int DisfigurementScore => (ActiveSplices.Count * 2) + (ActiveMalapties.Count * 5);

    public void AttemptSplice(TissueSample sample, int podTechLevel)
    {
        // Purity acts as a penalty: more mutated = harder to splice safely
        double successChance = (0.70 + (podTechLevel * 0.05)) * HumanDNAPurity;
        double roll = Random.Shared.NextDouble();

        if (roll <= successChance)
        {
            // Success!
            ActiveSplices.Add(sample.AssociatedMutation);
            HumanDNAPurity -= sample.GeneticWeight;
            GameLog.Add($"Splice successful! Your genome absorbs the {sample.Name}.");
        }
        else
        {
            // Failure - Mutate!
            Malapty newFault = MutationLibrary.GenerateCorrespondingMalapty(sample);
            ActiveMalapties.Add(newFault);
            HumanDNAPurity -= (sample.GeneticWeight * 0.5f); // Still corrupts the DNA
            GameLog.Add($"CRITICAL FAILURE: The DNA rejects the host. You have contracted {newFault.Name}!");
        }
    }
}

```

This completes the loop of your game's economy: money drives you to hunt monsters for the humans, but tissue harvesting drives you to hunt monsters for *yourself*. You are balancing on a knife-edge between being humanity's savior or its next cosmic boss.
